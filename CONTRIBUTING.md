# Contributing

Thanks for improving NewsCred RAG.

## Development setup

1. Fork or clone the repository.
2. Create a focused branch from `main`.
3. Install dependencies with `npm install`.
4. Start the service with `npm start`.

## Quality checks

Run `npm test` before opening a pull request. Add tests under `tests/` for retrieval, ranking, or API behavior changes. Use synthetic fixtures instead of sensitive source material.

## Pull requests

Describe the problem, implementation, evaluation impact, and validation. Never commit API keys, private documents, or generated dependency folders.
